========================================
  IRONFLOW — запуск на новом / другом ПК
========================================

1. УСТАНОВИТЕ PYTHON (если ещё нет)
   • Сайт: https://www.python.org/downloads/
   • Скачайте и установите Python 3.10 или новее
   • ВАЖНО: при установке поставьте галочку "Add Python to PATH"
   • Перезапустите компьютер после установки (или закройте все окна и откройте заново)

2. СКОПИРУЙТЕ ВСЮ ПАПКУ ПРОЕКТА
   • Папка ironflow-fitness целиком (со всеми файлами и подпапками)

3. ЗАПУСК
   • Дважды нажмите на start.bat
   • При первом запуске на этом ПК батник сам поставит нужные библиотеки (подождите 10–30 сек)
   • Откроется браузер с сайтом http://localhost:5000

Если не запускается:
   • Убедитесь, что Python установлен: откройте cmd и введите  python --version
   • Если "python не является внутренней командой" — переустановите Python с галочкой "Add to PATH"
   • Если порт 5000 занят — закройте другое приложение, которое его использует, или напишите в run.py другой порт (например 5001)
